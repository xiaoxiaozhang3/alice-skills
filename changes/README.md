# Changes

Each folder is one auditable change. Use one folder per task, feature, fix, or skill iteration.

## Naming

```text
YYYY-MM-DD-short-slug
```

## Required Files

- `00-change.md`
- `01-requirements.md`
- `02-design.md`
- `03-plan.md`
- `04-tasks.md`
- `05-progress.md`
- `06-verification.md`
- `07-review.md`
- `08-archive.md`

## Resume Rule

When resuming, read `05-progress.md` first, then `04-tasks.md`, then only the phase files needed for the current step.
