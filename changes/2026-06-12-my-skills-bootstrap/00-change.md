# Bootstrap My Skills Repository

## Summary

Create a personal engineering skill repository under `/Users/mac/Documents/obsidian/my-skills`.

## Source Request

The user wanted to generate a personal skill directory based on the Obsidian note `Agent skill 迭代式编写实战`, using selected material from `/Users/mac/Documents/obsidian/skills`, and borrowing governance ideas from Superpowers and OpenSpec.

## Scope

### In Scope

- Create repository structure
- Create engineering workflow skills
- Create fixed change folder format
- Add checklist and resume rules
- Add source mapping
- Add schema-first API rule to planning/rules

### Out Of Scope

- Installing skills into Codex
- Copying all upstream skills
- Creating a PM or marketing skill marketplace
- Adding scripts beyond templates in the first pass
